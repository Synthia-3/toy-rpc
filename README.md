客户端通过服务注册中心拿到一堆地址，该调哪个呢？最简单的方式，可以通过RR、WRR的方式去做LB。

如果做得更深入一些，可以从以下角度去优化：

根据服务实例的metrics做出动态调整, 比如响应时间等
利用一致性哈希， 提高本地缓存利用率
服务调用超时与重试： 在调用一个服务实例的时候，如果超时或者报错，怎么处理？
